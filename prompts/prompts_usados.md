#Documentación de Prompts utilizados

#Prompt 1 – Rol: Especialista en Extracción de Datos

**Contexto:**  
Se requiere extraer titulares de la sección “Elecciones” del medio La República.

**Objetivo:**  
Generar un script en Python que realice web scraping estático y almacene los datos en un archivo CSV estructurado.

#Prompt 2 – Rol: Especialista en Procesamiento y Limpieza

**Contexto:**  
Se cuenta con un dataset de titulares electorales en formato CSV.

**Objetivo:**  
Eliminar duplicados, normalizar texto, tokenizar titulares y generar variables útiles para el análisis.

#Prompt 3 – Rol: Analista de Datos y NLP

**Contexto:**  
Dataset limpio de titulares electorales previo a las Elecciones 2026.

**Objetivo:**  
Identificar agenda temática, palabras más frecuentes y analizar el sentimiento de los titulares para interpretar patrones de cobertura mediática.

#Prompt 4 – Rol: Arquitecto de GitHub y Documentación

**Contexto:**  
Proyecto colaborativo con scripts, datos y resultados de análisis.

**Objetivo:**  
Estructurar el repositorio, documentar la metodología y publicar los resultados mediante GitHub Pages.
